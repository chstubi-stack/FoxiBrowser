# 🦊 FoxiBrowser – Sicherer Kinder-Browser für Windows

Ein kindgerechter Desktop-Browser für Windows 10 & 11. Kostenlos, sicher und einfach zu bedienen.

**[⬇️ Jetzt herunterladen](https://github.com/chstubi-stack/FoxiBrowser/releases/latest)** · **[🌐 foxibrowser.de](http://foxibrowser.de)**

---

## Features

- 🛡️ **167.000+ blockierte Domains** – Adult, Glücksspiel, Fake News (Steven Black Liste)
- 🚫 **Werbeblocker** – 7 Filterlisten (EasyList, uBlock Origin, Fanboy u.a.)
- 🔒 **Eltern-PIN** – Adressleiste gesperrt, nur Eltern können neue Seiten aufrufen
- ⏰ **Nutzungszeit** – Tages-Limit festlegen, Wochendiagramm einsehen
- 👶 **Altersprofile** – Klein (3–6), Mittel (7–10), Groß (11–14) mit passendem Design
- ⭐ **Startseite** – Favoriten selbst zusammenstellen (YouTube Kids, KiKA, Maus u.v.m.)
- 🔄 **Auto-Update** – Neue Versionen werden automatisch im Hintergrund geladen
- 🪟 **Popup-Schutz** – Popups nur mit Eltern-PIN erlaubt
